# FE-base

## 기술 스택 (Tech Stack)

- **라이브러리:** React
- **빌드 툴:** Vite
- **스타일링:** Tailwindcss
- **컴포넌트 로직:** Headless UI
- **상태 관리:** Zustand
- **API 통신:** Axios
- **라우팅:** React Router

<br/>

## 요구 사항 (Requirements)

- Node.js 20.19+, 22.12+

<br/>

## 실행 방법 (How to run)

```bash
npm install
npm run dev
```
